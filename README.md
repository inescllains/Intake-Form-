# Introduction

Intake form to collect personal history and data from patients before they enter the doctor's office. 
The data is then stored in a csv file that can be stored in the organization's computers so that there is no risk of a breach of privacy. 
The form was done using JavaFX. 

## Instruction
To use this application, in addition to having java installed you also need to download the JavaFX jar files, and an opencsv jar file.
